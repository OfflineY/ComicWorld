# 📚 ComicWorld
一个漫画网站，它只是实现了最简单的功能，仿照着写的。Express + HTML + Jquery

## 🧾 目录

- [🎈 使用](#-使用)
- [📦 技术栈](#-技术栈)
- [📌 注意事项](#-注意事项)

## 🎈 使用

1. 克隆整个仓库
2. 安装依赖： npm i 或者 cnpm i 或者其他。
3. 双击 Comic.exe 运行。
4. 访问[http://localhost:3000/](http://localhost:3000/)即可。

> 更改端口详见注意事项

## 📦 技术栈

- Express
- Jquery
- Html + Css + Javascript

## 📌 注意事项

- 最近jsdelivr总是抽风，如果页面显示不正常打开public更改html的cdn源即可。
- 仅为学习娱乐，不适用于商用。
- 更改端口：更改 Main.js 开头即可。
